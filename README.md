在使用酸酸乳需要注意以下几点
--------------------------
1.在安装酸酸乳时请关闭杀毒软件，因为安装包使用winrar封装的，如果不关，很容易误报  
2.win10的用户建议使用ShadowsocksR-dotnet4.0，win10以下的用户建议使用ShadowsocksR-dotnet2.0  
3.如果没有ssr的话可以打开酸酸乳（普通版）------更新与下载-------下载ssr，然后在弹出的对话框选择是即可
